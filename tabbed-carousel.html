WAI-ARIA Design Pattern WAI-ARIA Term WAI-ARIA Definition HTML Implementation JavaScript Implementation Description
Tabs - Tabs are a set of layered sections of content, known as tab panels, that display one panel of content at a time. Each tab panel has an associated tab element, that when activated, displays the panel. The list of tab elements is arranged along one edge of the currently displayed panel, most commonly the top edge. - - -
Tabs Tabs or Tabbed Interface A set of tab elements and their associated tab panels. <div class="tabbed-carousel"> TabbedCarousel
Tabs Tab List A set of tab elements contained in a tablist element. <div role="tablist" aria-label="Choose photo to display">
Tabs tab An element in the tab list that serves as a label for one of the tab panels and can be activated to display that panel. <button role="tab" aria-controls="tabpanel-1" aria-selected="true" id="tab-1">
Tabs tabpanel The element that contains the content associated with a tab.
Carousel - The structure of a tabbed carousel is the same as a basic carousel except that: Each slide container has role tabpanel in lieu of group, and it does not have the aria-roledescription property.
Carousel - Features needed to provide sufficient rotation control include: Buttons for displaying the previous and next slides. <div class="tabpanels__nav">
Carousel - Features needed to provide sufficient rotation control include: Optionally, a control, or group of controls, for choosing a specific slide to display. For example, slide picker controls can be marked up as tabs in a tablist with the slide represented by a tabpanel element. <div role="tablist" aria-label="Choose photo to display">
